
# Django PDF AI Chatbot (LangChain + WebSocket)

A real-time AI-powered chatbot built with **Django**, **Channels**, and **LangChain**.  
It can read and answer questions from uploaded PDFs using **OpenAI embeddings**.

---

##  Features
- Upload and process PDFs for knowledge retrieval
- Real-time chat using WebSockets (Django Channels)
- Stores conversation history in SQLite
- Uses LangChain + OpenAI for intelligent responses


---

## Tech Stack
- **Backend:** Django, Django REST Framework
- **Real-time:** Django Channels, Daphne
- **AI / NLP:** LangChain, OpenAI API
- **Database:** SQLite
- **Vector Store:** FAISS
